Sobre Mim!

Olá! Sou uma estudante dedicada ao universo do desenvolvimento de jogos e criação web, buscando oportunidades para contribuir em projetos empolgantes. Tenho conhecimentos sólidos em HTML e CSS, utilizando-os para criar interfaces atraentes e funcionais. Meu domínio em Java Script ainda está em estágio básico, mas estou empenhado em aprimorá-lo. Além disso, possuo uma base no básico de Unity, C# e Python para o desenvolvimento de jogos, juntamente com habilidades na criação de jogos 2D.
